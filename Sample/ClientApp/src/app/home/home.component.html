Sample App

Welcome to Sample app for Management of Shipping containers. With this app, you can:

  • Add a new products to existing containers
  • Remove products from container

Add a Product then go to Manage Container to View/Remove them

Setup

Project developed with Sample.sln in VS2022. In VS, set Sample WebAPI project as startup and run. This will start building both the C# backend API host https://localhost:7298/ and the Angular `ClientApp`. The API host will redirect to ClientApp eventually at https://localhost:44422/

Frontend:

The app was build with the dotnet template for angular for a quick start, and therefore uses bootstrap styles. The app was build to try out Angular Material design.

Backend:

ASP.NET6.0 API.
